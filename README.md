# Login-Authentication-System
The Login Authentication System is a web-based security application developed using Python Flask and SQLite. The system allows users to register, log in securely, and access protected dashboard pages.

# Login Authentication System

## Project Overview

The Login Authentication System is a Flask-based web application that provides secure user registration and login functionality.

Passwords are securely hashed before storing in the database.


# Features

- User registration
- Secure login system
- Password hashing
- Session management
- SQLite database integration

# Technologies Used

- Python
- Flask
- SQLite
- HTML/CSS


# Installation

Install required libraries:

pip install flask werkzeug


# Run Project

python app.py
